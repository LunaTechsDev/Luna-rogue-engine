# Rogue Entities
> Rogue entities are enemies, items, pick ups on the Game Map. In the editor, they have special tags that signify what elements they are. These special tags link enemy and item information in the database with events that are placed on the map.

#### Example Tags
```js
<LREI:1> //Luna Rogue Engine Item Link

<LREE:1> //Luna Rogue Engine Enemy Link

<LREW:1> //Luna Rogue Engine Weapon Link

<LREA:1> //Luna Rogue Engine Armor Link
```
